Steward Scholar

Steward Scholar is a database website that maintains possible ethical issues with research papers.
Database is built on the assumption that an ethical issue raised for some paper will have causal effect on
linked research papers i.e the papers that cite the initial paper and so on.

Currently in development phase. The website provide following functionalities

  • Display Research Paper
    Search by Title. It will display information related to a research paper and flags associated with it
  • Add Flag
    On research paper display page, user can add a flag, consequently all papers reachable from the current
    will get flagged with proper intesity.
  • Display Flag
    On display research paper page, user can click view flag, user can view all the information regarding the flag and debates
  • Debate on Flag
    On display flag paper page, user can participate in debate regarding current flag, put forward counter arguments
  • Track Flag
    On flag display page, user can click track flag to find where it originated from.
